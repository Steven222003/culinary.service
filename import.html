📥 Импорт данных для кулинарного сервиса

Вставьте JSON с данными рецептов и нажмите "Импорт".

💡 Получите JSON на ПК через консоль (F12) командой:
console.log(JSON.stringify({recipes: JSON.parse(localStorage.getItem('recipes')), ratings: JSON.parse(localStorage.getItem('recipeRatings')), menu: JSON.parse(localStorage.getItem('cookMenu'))}));
