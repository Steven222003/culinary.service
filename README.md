# culinary.service
Кулинарный веб-сервис
Упрощенная версия приложения доступна по адресу : https://steven222003.github.io/culinary.service/.
Для получения последней актуальной версии приложения необходимо загрузить файл data.json из репозитория, перейти по адресу : https://steven222003.github.io/culinary.service/import.html и импортировать скаченный ранее файл, затем перейти на сам сервис : https://steven222003.github.io/culinary.service/.
